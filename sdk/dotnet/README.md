# ChainLogistics .NET SDK

.NET SDK for the ChainLogistics API with a client surface aligned to the Python, Rust, and Java SDKs.

## Included operations

- `HealthAsync()`
- `ReadinessAsync()`
- `LivenessAsync()`
- `MonitoringHealthAsync()`
- `DbHealthAsync()`
- `ListProductsAsync()`

## Build and test

```bash
dotnet test
```
