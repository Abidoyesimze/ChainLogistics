# ChainLogistics SDK Documentation

This directory contains shared SDK documentation for the Python, Rust, Java, and .NET SDKs.

## Documentation Index

- `INSTALLATION.md`: installation and setup guidance across supported SDKs.
- `API_REFERENCE.md`: high-level API reference by service and method.
- `EXAMPLES.md`: common end-to-end usage examples.
- `MIGRATION_GUIDE.md`: migration guidance for SDK upgrades.
- `python/README.md`: Python SDK details.
- `rust/README.md`: Rust SDK details.
- `java/README.md`: Java SDK details.
- `dotnet/README.md`: .NET SDK details.
- `benchmarks/benchmark_sdk.py`: benchmark smoke test for client caching behavior.

## Supported SDKs

- Python: `sdk/python`
- Rust: `sdk/rust`
- Java: `sdk/java`
- .NET: `sdk/dotnet`
